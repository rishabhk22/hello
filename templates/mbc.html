Problem & Context

Runway Debris Detection & Autonomy Challenges

Runway Foreign Object Debris (FOD) poses critical safety risks to aircraft operations. Traditional inspection methods are time-intensive, labor-dependent, and can miss dangerous objects. In high-traffic airports, limited inspection windows create operational bottlenecks impacting throughput and reliability.

This challenge aligns with broader autonomy problems: deploying scalable, reliable autonomous systems that operate safely in dynamic environments. The solution requires multi-agent coordination, robust perception, and fail-safe mechanisms—critical for real-world autonomy applications.

MBC in One Line

An autonomous UAV swarm system that uses computer vision and coordinated flight patterns to detect and report Foreign Object Debris on runways, enabling rapid, reliable inspection without disrupting airport operations.

Project Description

High-Level Solution

A coordinated multi-UAV swarm architecture with advanced computer vision to autonomously inspect runways for Foreign Object Debris. Combines autonomous flight coordination, real-time FOD detection, and comprehensive coverage patterns for runway safety.

System Architecture

UAV Swarm Architecture

A coordinated multi-UAV architecture where drones operate autonomously while maintaining swarm-level coordination. Each UAV uses computer vision sensors and communicates with a central system for comprehensive runway coverage.

  • Swarm coordination: Manages flight paths, safe separation, and dynamic re-planning
  • Computer vision: Real-time FOD detection and classification
  • Sensor suite: RGB cameras, depth sensors, and IMU data
  • Ground control: Aggregates results and provides operator interface

Mission Flow

Inspection Workflow

The inspection workflow follows a systematic approach:

  • Pre-flight planning: Generate optimal coverage patterns based on runway dimensions
  • Autonomous deployment: Coordinated takeoff and swarm formation positioning
  • Systematic coverage: Grid-based inspection with real-time path adaptation
  • Real-time detection: On-board processing and transmission to ground control
  • Result aggregation: Compile detections, validate findings, generate reports
  • Safe recovery: Coordinated landing sequence without conflicts

Computer Vision & Autonomy

FOD Detection & Classification

Advanced computer vision algorithms process real-time imagery to identify foreign objects of varying sizes and materials. Multi-scale detection techniques catch both large debris and small objects that could damage aircraft.

  • Multi-scale detection: Identifies objects from small screws to large debris
  • Material classification: Distinguishes metal, plastic, rubber, and organic materials
  • Risk assessment: Evaluates objects for potential aircraft damage risk
  • Real-time processing: On-board computation for immediate detection
  • Adaptive learning: Improves accuracy through feedback and continuous learning

UI/UX

Operator Interface

A simple mission view that surfaces critical information without overwhelming operators:

  • Real-time swarm status: Live view of UAV positions and flight paths
  • Detection alerts: Immediate notifications with location and risk assessment
  • Mission progress: Visual indicators for coverage completion
  • Override controls: Options to confirm or override automated decisions
  • Audit trail: Complete log of detections and system actions

Focus on operational clarity and auditability—keeping human oversight lightweight but effective.
